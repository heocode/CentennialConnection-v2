# PetApp

> A production-inspired full-stack mobile application built from scratch to explore modern software engineering practices and backend development.

## About the Project

PetApp is a learning project created to simulate the development of a real production application.

The primary goal is not simply to build a chat application, but to understand how modern software is designed, implemented, deployed, and maintained. Every major stage of the development process is documented, from the initial product vision to the final deployment.

Instead of relying on backend-as-a-service platforms, this project focuses on building the core components from scratch, including authentication, database design, REST APIs, real-time communication, deployment, and infrastructure.

The project is developed as if it were intended for production, following common software engineering practices such as documentation, version control, modular architecture, and incremental development.

---

## Project Goals

- Learn backend development using Node.js and TypeScript
- Design a scalable application architecture
- Build secure REST APIs
- Design and manage a PostgreSQL database
- Implement JWT authentication
- Build real-time communication using WebSockets
- Containerize services with Docker
- Deploy the application on a Linux server
- Practice production-oriented software development
- Document the complete engineering process
